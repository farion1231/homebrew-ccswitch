# Homebrew Tap for CC Switch

> ⚠️ **DEPRECATED**: This tap has been superseded by the official Homebrew cask. Please use `brew install --cask cc-switch` directly without adding this tap.

### 🌐 The Only Official Website: **[ccswitch.io](https://ccswitch.io)**

This tap distributes [CC Switch](https://ccswitch.io), the cross-platform desktop manager for Claude Code, Codex, OpenCode, OpenClaw, Gemini CLI & Hermes Agent.

## Installation

**Recommended**: Install directly from the official Homebrew cask:

```bash
brew install --cask cc-switch
```

<details>
<summary>Legacy installation via this tap (not recommended)</summary>

```bash
brew tap farion1231/ccswitch
brew install --cask cc-switch
```

</details>

## Updating

```bash
brew update
brew upgrade --cask cc-switch
```

## Uninstallation

```bash
brew uninstall --cask cc-switch
brew untap farion1231/ccswitch
```
